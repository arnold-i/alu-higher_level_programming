# python-if_else_loops_functions

Introductory Python project on conditionals (if / elif / else),
loops (for / range), and functions. Part of the ALU
alu-higher_level_programming curriculum.

## Requirements

- Ubuntu 20.04 LTS
- Python 3.x (python3)
- All files end with a new line
- First line of every file: #!/usr/bin/python3
- Code follows pycodestyle (PEP 8)
- All files are executable

## Tasks

- 0-positive_or_negative.py : Prints whether a random number is positive, negative, or zero.
- 1-last_digit.py : Prints the last digit of a random number and how it compares to 5/0.
- 2-print_alphabet.py : Prints the lowercase ASCII alphabet, no newline.
- 3-print_alphabt.py : Prints the lowercase alphabet except q and e.
- 4-print_hexa.py : Prints 0 to 98 in decimal and hexadecimal.
- 5-print_comb2.py : Prints numbers 0 to 99, comma-separated, two digits.
- 6-print_comb3.py : Prints all smallest combinations of two different digits.
- 7-islower.py : islower(c) returns True if c is lowercase.
- 8-uppercase.py : uppercase(str) prints a string in uppercase.
- 9-print_last_digit.py : print_last_digit(number) prints and returns the last digit.
- 10-add.py : add(a, b) returns a + b.
- 11-pow.py : pow(a, b) returns a to the power of b.
- 12-fizzbuzz.py : fizzbuzz() prints the classic FizzBuzz from 1 to 100.
