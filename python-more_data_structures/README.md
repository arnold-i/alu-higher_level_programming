# python-more_data_structures

Project on Python sets, dictionaries, list comprehensions, and higher-order
functions like map/filter/lambda. Part of the ALU
alu-higher_level_programming curriculum.

## Requirements

- Ubuntu 20.04 LTS
- Python 3.x (python3)
- All files end with a new line
- First line of every file: #!/usr/bin/python3
- Code follows pycodestyle (PEP 8)
- No imported modules
- Function tasks are imported via a *-main.py driver

## Tasks

- 0-square_matrix_simple.py : Returns a new matrix with each value squared.
- 1-search_replace.py : Returns a new list with all occurrences replaced.
- 2-uniq_add.py : Adds all unique integers in a list.
- 3-common_elements.py : Returns the intersection of two sets.
- 4-only_diff_elements.py : Returns the symmetric difference of two sets.
- 5-number_keys.py : Returns the number of keys in a dictionary.
- 6-print_sorted_dictionary.py : Prints a dictionary by ordered keys.
- 7-update_dictionary.py : Adds or replaces a key/value in a dictionary.
- 8-simple_delete.py : Deletes a key from a dictionary if present.
- 9-multiply_by_2.py : Returns a new dict with all values multiplied by 2.
- 10-best_score.py : Returns the key with the biggest integer value.
- 11-multiply_list_map.py : Multiplies list values by a number using map (max 3 lines).
- 12-roman_to_int.py : Converts a Roman numeral string to an integer.
