# python-exceptions

Project on raising, catching, and handling exceptions in Python using
try / except / else / finally. Part of the ALU
alu-higher_level_programming curriculum.

## Requirements

- Ubuntu 20.04 LTS
- Python 3.x (python3)
- All files end with a new line
- First line of every file: #!/usr/bin/python3
- Code follows pycodestyle (PEP 8)
- No imported modules
- Function tasks are imported via a *-main.py driver

## Tasks

- 0-safe_print_list.py : Prints x elements of a list; returns the real number printed (no len()).
- 1-safe_print_integer.py : Prints a value as an integer; returns True/False (no type()).
- 2-safe_print_list_integers.py : Prints first x elements that are integers, skips others.
- 3-safe_print_division.py : Divides two integers, prints result in finally, returns it or None.
- 4-list_division.py : Divides two lists element by element with full error handling.
- 5-raise_exception.py : Raises a TypeError.
- 6-raise_exception_msg.py : Raises a NameError with a custom message.
